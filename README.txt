
Produk Kampung Lalang - PWA (plain HTML/CSS/JS)
---------------------------------------------
To run locally:
  1. cd to the project folder
  2. serve with a static server (example: npx serve . or python -m http.server 8080)
  3. open http://localhost:5000 (or the provided port)
Notes:
  - This is a demo PWA. Data (products & cart) are stored in localStorage for demonstration.
  - Admin UI is accessible via the 'Admin' button in header.
